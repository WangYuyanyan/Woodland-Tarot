
## 核心功能模块分析

* **UI 与 3D 动画**：使用 CSS3 `transform: preserve-3d` 实现的轮播图阵列和卡牌翻转效果；新艺术（Art Nouveau）风格的 UI 设计。
* **手势视觉交互**：引入 `@mediapipe/hands` 库，通过摄像头捕捉手部关键点（判断握拳 `isFist` 和张开手掌 `isOpenPalm`）。
* **Canvas 背景系统**：使用 HTML5 Canvas 绘制动态星空、星座连线、萤火虫和环境雾气。
* **AI 大模型对话流**：使用 `fetch` 接口对接 DeepSeek API，并实现流式（Streaming）数据接收与简单的 Markdown 解析。
* **Web Audio API 音效**：不依赖外部音频文件，纯用代码合成（OscillatorNode）选择、翻牌和魔法音效。

### 该代码实现的功能概括

简而言之，这是一个**沉浸式的 AI 塔罗牌占卜 Web 应用**。它的主要功能包括：

1. **非接触式手势操控**：用户无需鼠标键盘，对着摄像头挥动手部即可控制卡牌滚动，通过“握拳”动作抽取代表“过去、现在、未来”的三张牌，通过“张开手掌”触发命运解读。
2. **拟真的物理与视觉反馈**：实现了 3D 视角的卡牌阵列、翻牌动画，并配有鼠标/手部位置跟随的动态卡牌光照效果（Light Tracing）和粒子飞散特效。
3. **AI 深度解牌**：允许用户输入心中的困惑，结合抽出的三张塔罗牌的正逆位，调用 DeepSeek大模型，以荣格心理学和叙事疗法的口吻进行具有同理心的个性化流式解读。
4. **独立运行的沉浸环境**：背景包含自动生成的代码级星空和萤火虫动画，同时内置了纯代码合成的交互音效，无需加载庞大的外部媒体文件。
5. **往昔典藏（历史记录）**：自动将用户的提问、抽牌结果和 AI 解读文本保存在本地浏览器缓存中，方便随时回顾。

## 手动修改帮助note：

#### 静态 UI 文字（在 HTML 部分寻找）：

你可以直接在代码的上半部分（`<style>` 标签之后，`<script>` 标签之前的 HTML 结构中）找到并修改这些内容：

* **网页标签页标题**：`<title>命运塔罗 · Fatum Arcana</title>`
* **加载界面的标题**：`<div id="loader-title">命运塔罗</div>` 和 `<div id="loader-sub">FATUM ARCANA</div>`
* **主界面的顶部标题**：`<div class="header-title">命运塔罗</div>` 和 `<div class="header-sub">Fatum Arcana · 三牌时间阵</div>`
* **按钮文字**：如 `<button id="start-btn">✦ 开始体验</button>`
* **手势操作提示区**：`<div class="zone-label">◁ 向左探索</div>`
* **各个弹窗的标题**：
* 解读弹窗：`<div class="interp-title">✦ 命运启示录 ✦</div>`
* 提问弹窗：`<div class="qm-title">✦ 问卜天机 ✦</div>`
* 历史记录弹窗：`<div class="hm-title">◎ 往昔牌阵典藏</div>`
* 设置弹窗：`<div class="settings-title">⚙ API 密钥配置</div>`



#### 动态提示文字（在 JavaScript 部分寻找）：

这些文字会根据用户的操作动态改变，你需要在 `<script>` 标签内的逻辑中修改：

* **加载动画的轮播提示语**：搜索 `const msgs=['正在编织星轨...','汇聚神秘能量...'` 这个数组。
* **状态栏提示语（`setStatus` 调用的地方）**：搜索 `setStatus('握拳选中你的命运之牌 (1/3)')`、`setStatus('命运揭示...')` 等。
* **AI 的系统提示词（Prompt）**：如果你想修改 AI 解读的语气和排版，搜索 `const systemPrompt = ...` 和 `const outputFormat = ...`。
* **塔罗牌的名称**：搜索 `const MAJOR = [` 和 `const SUITS = [` 数组，里面包含了所有卡牌的中文名（如 `n:'愚者'`）和英文名。
